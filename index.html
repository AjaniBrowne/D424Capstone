Student Tracker App

Download the app:

This .NET MAUI application is a student term planner that lets you log in and then manage academic terms, courses within each term, and objective/performance assessments tied to each course. It persists data locally using SQLite, enforces key validation rules (no duplicates, required fields, and date ranges where course dates must fall within the selected term and assessment dates must fall within the selected course), and supports searching terms/courses/assessments with results displayed in CollectionView lists. It also schedules local notifications for course and assessment start/end dates and can generate an on-screen term report containing a title, a date-time timestamp, and multi-column/multi-row sections for terms, courses, and assessments.
